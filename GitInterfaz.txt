Repaso GitHub y Visual Studio Code

1. Clonar el repositorio de la clase anterior.
	$ git clone <url repositorio>
2. Abrir Visual Studio Code.
3. Abrir la terminal de Visual Studio.
	Control + ñ
4. Crear un archivo y subirlo a GitHub.
5. Crear una rama desde Visual Studio.
	$ git checkout -b <branch>
6. Crear un archivo y subirlo a GitHub.
8. Hacer merge entre master y la nueva rama. 
7. Crear una rama desde GitHub.
8. En Visual Studio hacer git pull. 
9. Ir a la nueva rama.
10. Crear un archivo y subirlo a GitHub. 
11. Hacer merge entre master y la nueva rama. 
12. En Visual Studio hacer git pull. 
13. Ir a master.
14. Editar uno de los archivos ya creados.
15. Subir los cambios a GitHub.
16. Ir a la rama creada en 7. 
17. Editar el mismo archivo que se editó en 14.
18. Subir los cambios a GitHub.
19. Hacer merge.
